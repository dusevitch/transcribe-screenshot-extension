QUICK SETUP GUIDE - Transcribe Images Extension
================================================

Super simple 3-step setup! No OAuth or complex configuration needed.

1. GET GEMINI API KEY
   - Go to: https://makersuite.google.com/app/apikey
   - Click "Create API Key"
   - Copy it (you'll enter this in the extension settings)

2. LOAD EXTENSION
   - Download or clone: https://github.com/dusevitch/transcribe-images-extension
   - Open Chrome and go to: chrome://extensions/
   - Turn ON "Developer mode" (top right toggle)
   - Click "Load unpacked"
   - Select the "transcribe-images-extension" folder

3. CONFIGURE EXTENSION
   - Click the extension icon in Chrome toolbar
   - Click "Settings" at the bottom
   - Paste your Gemini API Key
   - Click "Save Settings"

THAT'S IT! You're ready to go.

HOW TO USE
==========

1. Click extension icon
2. Click "Capture Screenshot" and select area
3. Repeat for multiple screenshots (optional)
4. Click "Transcribe with Gemini"
5. Copy formatted text OR download as Word/HTML/PDF/Text

FEATURES
========

✓ Rich text output (bold, italics, tables, colors preserved)
✓ Copy with formatting - paste into Word, Google Docs, Notion
✓ Download as .docx, .html, .pdf, or .txt
✓ Image thumbnails - see what you captured
✓ Multiple screenshots - capture several images in order
✓ Editable output - fix mistakes before saving

UPDATING
========

To get latest features:
1. cd transcribe-images-extension
2. git pull origin main
3. Go to chrome://extensions/
4. Click refresh icon on the extension

No Chrome Web Store needed - you control updates!
